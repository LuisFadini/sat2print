# sat2print

A simple CLI tool that takes a College Board SAT Question Bank PDF and converts it into a compact, print-friendly format.

It removes unnecessary whitespace, skips blank pages, and packs multiple questions per page, making it easier and cheaper to print.

## Feature Roadmap

- [x] Extract pages
- [x] Crop pages to content
- [x] Combine processed pages into a printable PDF
- [x] Allow specifying input file via CLI
- [ ] Support multiple output formats (PDF, PNG, per-question images)
